Sistema web para logística de productos

Proyecto CRUD académico para gestionar inventario, ubicación física y compra simulada de productos en bodega.

  • Administrador: registra, edita, elimina y ajusta stock.
  • Cliente: consulta productos y simula compras.
  • Ubicación completa: bodega, zona, torre, estantería, módulo y posición.
  • Persistencia local con localStorage.

Usuarios de prueba: admin/admin y cliente/cliente.
